Onboarding tool for large companies | Endava

Onboarding Manager Experience

Several dependencies on their employee training upon stakeholders from other company areas

The Opportunity - Endava BOD expansion has rose up to hundreds of employees that require an onboarding process that can be verified and tracked.

The solution -A task manager product was developed for new joiners and admins that boosts the integration of every single process and stakeholder during the new joiner onboarding. The tool is made to work on any training situation, for instance: technical trainings.
This product has iterated 4 times since it's conception on every user test.

The UX Approach -The solution was conceived and refined by UX Research and prototyping using these qualitative and quantitative methods:

1. Task Analysis
Stakeholder Interviews & Benchmarking Journey mapping for all stakeholders
2. Scoping
Quantitative - Tasks count, Dependencies flow Qualitative - Reviewal of 4 self-methods used currently for training tracking by each stakeholder, most of them made on Excel
3. Prototyping
User Test - User cases derived from research tried on Wireframes Quick Iteration - Several versions were deployed during user tests to accelearte the error making prior to development
4. Deployment
User Stories, Wireframing, Cases of Use

The Deliverables - Customer Journey Map, Stakeholder Map, User Persona Chart, Current User Flow, Wireframes
